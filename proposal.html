Capstone Project Proposal

User Authentication System with Role-Based Access Control

Overview

This project designs and develops a secure authentication system reflecting real-world cybersecurity practices. Users can register and log in securely with role-based access control.

Problem Statement

Many applications require controlled access to protect data and system functionality. Without authentication and access control, systems are vulnerable to unauthorized use.

Project Objectives

  • Secure login and logout system
  • Password hashing using bcrypt
  • Role-Based Access Control (RBAC)
  • Separate admin and standard user functionality
  • Organized and secure system design

Tools Used

  • Python
  • MySQL
  • bcrypt
  • VS Code
